# E2E Test Suite for Northwind Goods

This directory contains the End-to-End (E2E) automation suite for the Northwind Goods storefront, built using **Playwright**.

## Structure

The test suite is structured to separate normal user behavior from specific edge cases (bugs) observed during exploratory testing.

### 1. Core User Journeys
These tests verify that the main application features work correctly. They should all **pass**.
- `browsing.spec.ts`: Homepage, product listing, search, and category filtering.
- `product-detail.spec.ts`: Viewing product pages, checking stock status, and selecting options.
- `cart.spec.ts`: Adding/removing items, updating quantities, applying promo codes.
- `auth.spec.ts`: Login, registration, and logout.
- `checkout.spec.ts`: Completing an order end-to-end.

### 2. Anomalies & Observations
Based on the [Exploratory Report](../docs/exploration-report.pdf), we have a dedicated file that tests specifically for the known bugs.
- `anomalies.spec.ts`: Contains specific edge cases:
  - Footer category links (Missing link and URL sync bug)
  - Cart price calculation bug
  - Account menu gap bug (Desktop hover issue)
  - Order details viewability observation

> [!WARNING]
> The tests in `anomalies.spec.ts` are designed to **FAIL** because they assert the correct behavior against the known bugs in the application. This proves the tests are valid and the bugs exist.

## Setup and How to Run

### Prerequisites

Ensure you have Node.js and `pnpm` installed. The Northwind Goods application must be running locally for the tests to execute against it.

### 1. Install Dependencies

In the root directory of the project:

```bash
pnpm install
```

Install the required Playwright browsers:

```bash
pnpm exec playwright install --with-deps chromium
```

### 2. Start the Application

Start the local development server:

```bash
pnpm dev
```

The application should be running at `http://localhost:5173`.

### 3. Run the E2E Tests

In a new terminal window, execute the test suite:

```bash
pnpm exec playwright test
```

To run only the core tests (which should pass):
```bash
pnpm exec playwright test tests/browsing.spec.ts tests/product-detail.spec.ts tests/cart.spec.ts tests/auth.spec.ts tests/checkout.spec.ts
```

### 4. View the Report

After the tests finish running, view the HTML report generated by Playwright:

```bash
pnpm exec playwright show-report
```
